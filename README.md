# Space Sim

![ci status](https://github.com/wolfadex/space-sim/actions/workflows/ci.yml/badge.svg)

⚠️ WIP ⚠️

A 4X space game.

## Features

- Galaxy generation
  - Solar system generation
    - Star generation
    - Planet generation

## Get Started

### Development

```bash
npm install
npm run dev
```

```bash
npm run review-watch
```

```bash
npm run test-watch
```

### Production

```bash
npm ci
npm run build
```
