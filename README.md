# Space Sim

![pr status](https://github.com/wolfadex/space-sim/actions/workflows/test-pr.yml/badge.svg)

⚠️ WIP ⚠️

A 4X space game inspired by Dwarf Fortress.

<img src="./Screenshot 2022-01-17.png" />

## Features

- Galaxy generation
  - Solar system generation
    - Star generation
    - Planet generation
- Civilizations, with:
  - Population size
  - Reproduction rate
  - Names
  - Happiness
  - Knowledge (can be gained, not yet lost)
  - Logs of important events

## Get Started

### Development

```bash
npm install
npm run dev
```

```bash
npm run review-watch
```

```bash
npm run test-watch
```

### Production

```bash
npm ci
npm run build
```
